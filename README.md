"# Spotify Clone"

A front-end clone of the Spotify web player built with plain HTML and CSS. The project recreates the look and feel of Spotify's desktop interface, including the sidebar, content cards, sticky navigation, and bottom music player.

## Features

- Sidebar navigation with Home, Search, and Library sections
- Featured playlist and chart cards in the main content area
- Sticky top navigation for quick access actions
- Fixed music player bar with playback, progress, and volume controls
- Responsive layout behavior for smaller screens

## Tech Stack

- HTML5
- CSS3
- Font Awesome for icons
- Google Fonts (`Montserrat`)

## Getting Started

This is a static site, so there is no build step or package installation required.

1. Clone the repository.
2. Open the project folder in VS Code or your preferred editor.
3. Open `index.html` in a browser, or use a local server such as VS Code Live Server.
